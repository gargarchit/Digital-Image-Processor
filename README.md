#Digital Image Processor

A QT desktop application for image processing.

##Features to Implement

* Crop
* Grayscale
* Negative
* Sepia
* Hue
* Salt and Pepper
* Stretch
* Warp
* Rotate
* Invert
* Gaussian blur
* Quick Blur
* Contrast Enhancement (Histogram Equalization)
* Local Contrast Enhancement
* Add Watermark
* Sobel Edge detection
* Halftoning
* Dither
